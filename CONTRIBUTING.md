# Contributing to Awesome AI Agent Verification

Thanks for contributing! Here's how to add a tool or resource.

## Adding a New Tool

1. Fork this repository
2. Add your tool to the appropriate section in `README.md`
3. Follow the existing table format
4. Include: name, link, pricing, key features, limitations
5. Submit a Pull Request

## Requirements

- Tool must be related to phone verification for AI agents or automated systems
- Include accurate pricing information
- Note any limitations (VoIP detection, geographic restrictions, etc.)
- Links must be working and point to the official site
- No affiliate links

## Format

For tools/services:
```markdown
| [Tool Name](https://example.com) | $X/mo or per-use | Feature | Notes |
```

For articles:
```markdown
- [Article Title](https://example.com) - Brief description
```

## Quality Standards

- Descriptions should be factual, not promotional
- Include both strengths and limitations
- Pricing should be current and verifiable
- Categories should be accurate (Real SIM vs VoIP)

## Questions?

Open an issue if you're unsure where something belongs!
